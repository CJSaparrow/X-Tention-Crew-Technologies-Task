To run this project in ubuntu use the below given steps:

To set-up the environment run the below mention commands:

1) sudo apt install python3-venv python3-dev -y
2) python3 -m venv my_TensorFlow_venv
3) source my_TensorFlow_venv/bin/activate
4) cd my_TensorFlow_venv/

To download and extract the project from github use below comand:

5) wget https://github.com/CJSaparrow/X-Tention-Crew-Technologies-Task/archive/refs/heads/main.zip
6) sudo apt-get install unzip
7) unzip main.zip
8) cd X-Tention-Crew-Technologies

To install all the required libraries used in project run the below command:

9) pip install -r /my_TensorFlow_venv/X-Tention-Crew-Technologies-Task-main/requirements.txt -f file:/my_TensorFlow_venv/X-Tention-Crew-Technologies-Task-main/

Now to run the code use the below command:

10) python NESI_LSTM.py

To visual the prediction output follow the bellow commands:

11) sudo apt install feh
12) feh Output.png
